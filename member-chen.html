HTML学习笔记

2025.4.30 - 搭建静态网站 部署Tomcat

静态网站由 HTML、CSS 等固定文件构成，可用 VSCode 搭建。先创建项目文件夹，编写小组及成员主页代码， 利用 Live Server 插件预览，还能通过部署到 GitHub Pages 等让他人访问。

安装 Tomcat 可部署网页。安装后启动，将网页文件放至 webapps 目录。若端口被占， 可用命令行或任务管理器查看并终止进程，还可修改 Tomcat 默认端口避免冲突。

2025.5.7 - CSS布局实践

尝试用flex布局实现小组主页的左右分栏，解决了图片与文字垂直对齐的问题：

            .header {
                display: flex;
                align-items: center;  /* 垂直居中对齐 */
                justify-content: space-between;  /* 内容两端分布 */
            }
