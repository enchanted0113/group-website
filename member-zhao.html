HTML学习笔记

2025.4.30 - HTML整体认知

HTML（超文本标记语言）用于构建网页结构，它能定义内容的语义（如标题、段落）、嵌入资源（图片/链接/视频），并为浏览器和搜索引擎提供页面信息。以下是最基础的HTML5文档结构示例：

<DOCTYPE html>  <-- 声明文档类型为HTML5 -->
<html lang="zh-CN">  <-- 根标签，lang指定语言 -->
<head>  <-- 头部信息，不直接显示在页面 -->
    <meta charset="UTF-8">  <-- 设置字符编码 -->
    <title>我的第一个网页</title>  <-- 页面标题 -->
</head>
<body>  <-- 页面主体内容 -->
    <h1>欢迎来到我的网页</h1>  <-- 一级标题 -->
    <p>这是一个段落</p>  <-- 段落内容 -->
</body>
</html>

这个示例包含了HTML5的最小必要结构，浏览器会根据这些标签解析并渲染页面。

2025.5.2 - HTML标记和属性基础

HTML标记（标签）通常成对出现（如<p>...</p>），部分为单标签（如<img/>）。属性用于补充标签的额外信息，格式为"属性名=属性值"（值需用引号包裹）。以下是常见标签的示例：

1. 文本类标签

<h2>二级标题</h2>  <-- 标题标签（h1-h6，字号递减） -->
<p>这是一个普通段落，<b>加粗文字</b>，<em>强调文字</em></p>
<-- 输出：这是一个普通段落，加粗文字，强调文字 -->

2. 列表标签

<h3>无序列表（项目符号）</h3>
<ul>  <-- 无序列表（unordered list） -->
    <li>地理信息科学</li>  <-- 列表项（list item） -->
    <li>Python编程</li>
</ul>

<h3>有序列表（数字序号）</h3>
<ol>  <-- 有序列表（ordered list） -->
    <li>安装VS Code</li>
    <li>编写HTML代码</li>
</ol>

3. 链接与图片

<-- 链接标签（a）：href指定目标地址，target="_blank"表示新窗口打开 -->
<a href="https://www.douban.com" target="_blank">访问豆瓣</a>

<-- 图片标签（img）：src指定图片路径，alt为图片描述（加载失败时显示） -->
<img src="zhaoyuhan.jpg" alt="赵语涵个人照" width="150">

2025.5.3 - HEAD元素学习

HEAD标签内的内容不直接显示在页面，但对页面功能和优化至关重要。以下是常用元素的代码示例：

1. meta标签：定义元信息

<-- 字符编码（必须！否则可能乱码） -->
<meta charset="UTF-8">

<-- 视口设置（移动端适配关键） -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<-- 页面描述（供搜索引擎抓取，提升SEO） -->
<meta name="description" content="赵语涵的个人主页，包含地理信息科学学习笔记">

<-- 关键词（已逐渐被搜索引擎弱化，但仍可补充） -->
<meta name="keywords" content="地理信息科学,HTML学习,个人主页">

2. link标签：引入外部资源

<-- 引入外部CSS样式表（让页面更美观） -->
<link rel="stylesheet" href="styles.css">

<-- 引入网站图标（浏览器标签栏显示的小图标） -->
<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">

3. script标签：嵌入JavaScript

<-- 直接嵌入JS代码（通常放head或body末尾） -->
<script>
    alert("欢迎访问我的主页！");  <-- 弹出提示框 -->
</script>

<-- 引入外部JS文件（例如交互逻辑） -->
<script src="main.js"></script>
