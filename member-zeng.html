HTML学习笔记

2025.4.30 HTML文档基本结构

学习了根元素与基础标签：
<!DOCTYPE html>
<html lang="zh-CN"> <!-- lang属性指定页面语言，方便搜索引擎和辅助工具识别 -->
<head>
  <meta charset="UTF-8"> <!-- 设置字符编码，常见UTF-8可支持多语言字符 -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 移动端适配，确保页面在不同设备正确显示 -->
  <title>静态网页标题</title> <!-- 页面标题，显示在浏览器标签页 -->
</head>

2025.5.1 核心标签与应用

1.文本与段落标签
（1）标题标签：<h1>-<h6>用于定义不同层级标题，搜索引擎会根据标题层级抓取内容重点；
（2）段落标签：<p>包裹文本形成段落，<br>用于强制换行，<hr>插入水平分隔线。

2.多媒体元素
（1）图像标签：<img src="image.jpg" alt="描述文字" width="300" height="200">，src指定图片路径，alt用于图片加载失败时显示替代文字，width和height设置图像尺寸；
（2）音频 / 视频：HTML5 支持原生多媒体播放。

3.链接与导航
（1）超链接：<a href="https://www.example.com" target="_blank">访问示例网站</a>，target属性可设置链接打开方式，_blank为新窗口打开；
（2）导航栏：结合<nav>标签与列表标签构建。

2025.5.4 属性与样式基础

1.常用属性
class与id：class用于批量选择元素应用样式（如.btn { color: white; }），id为元素唯一标识（如#logo { width: 100px; }）；
style内联样式：直接在标签内定义样式，如<p style="color: red; font-size: 16px;">红色文本</p>。

2.基础样式设置
通过<style>标签在<head>内添加 CSS 样式，或链接外部样式表（<link rel="stylesheet" href="styles.css">）。

            .header {
                display: flex;
                align-items: center;  /* 垂直居中对齐 */
                justify-content: space-between;  /* 内容两端分布 */
            }
